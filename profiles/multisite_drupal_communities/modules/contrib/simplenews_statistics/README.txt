DESCRIPTION
-----------
Simplenews Statistics gathers newsletter statistics like the open rate and CTR
(click-through rate) and presents the information to administrators through an
easy to use interface.

REQUIREMENTS
------------
 * Simplenews.
 * Views.
 * The ability to send HTML email.

INSTALLATION
------------
 1. Install and enable this module.
 2. Configure settings at admin/config/simplenews/statistics.
 3. View statistics for individual newsletters at node/%/simplenews_statistics.
