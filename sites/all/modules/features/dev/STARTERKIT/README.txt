Readme file for the Features STARTERKIT For multisite
------------------------------------------------------

This folder serves as a starter kit to create your own feature for the multisite.
It contains the basic structure of a feature as well as a test file to help you ensure your scripts respects the Multisite guidelines as set here:

https://webgate.ec.europa.eu/CITnet/confluence/display/MULTISITE/Custom+features

Features dependencies
------------------------------------------------------

When a feature is generated, check if feature has not include unwanted dependencies. If it has, the dependencies must be removed manually in the .info file .
